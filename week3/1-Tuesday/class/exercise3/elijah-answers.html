Here's what I like about lists:

  • That's about it

When you make a list, do it in this order:

  1. Decide if it's an ordred list or not
  2. If it's not, use the ul tag, otherwise use the ol tag.
  3. Use li tags nested in the list for your list items.
      4. Curveball! You can nest lists inside each other
  4. You've got this!
