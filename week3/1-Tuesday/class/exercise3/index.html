EXERCISE 3: LISTS

This exercise is about making lists
Here's what I like about lists:

  • They're useful for displaying lists of stuff
  • Lists are great
  • Can be ordered or unordered, but always use the "li" tag within the list

When you make a list, do it in this order:
  1. Decide if its an ordered list or not.
  2. "Ul" tag for unordered list and "ol" tag for ordered list
    • you can nest lists inside other lists, can mix ul and ol
      1. Back to ordered list
  3. Go back to the main list
